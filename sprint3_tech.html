.Describe what the DOM is and an example of how you might interact with it.

The DOM can be described as an interface for HTML files which represents the complex document structure, style, content, and much more information. The DOM works and represents the files and documents as elements or nodes, and because of this, we can connect to the page through programming languages. The DOM can be manipulated through Javascript.


.Explain the difference between accessing data from arrays and objects.

In JavaScript, everything is an object. Objects are defined by their properties (data) and types. Those properties will define the object characteristics. We know that variables are like containers for data values. Arrays allows us to access to our values in an specific order, and also can store different or multiple values in just a single variable.
